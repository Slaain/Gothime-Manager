USER
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a GET method : http://localhost:4000/api/users?email=XXX&username=YYY -> curl "http://localhost:4000/api/users?email=leon2003@turner.name&username=manuel1949"
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a GET method : http://localhost:4000/api/users/:userID -> curl http://localhost:4000/api/users/6
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a POST method : http://localhost:4000/api/users -> Invoke-RestMethod -Uri http://localhost:4000/api/users -Method Post -ContentType "application/json" -InFile "user.json"
Fichier json associé : 
{
  "user": {
    "email": "milo@gmail.com",
    "username": "milo484"
  }
}
