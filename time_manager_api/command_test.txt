
-----------------------------------------------------------------------------------------------------------------------------------------------------------
USER
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a GET method -- MARCHE
http://localhost:4000/api/users?email=XXX&username=YYY
Commande : curl "http://localhost:4000/api/users?email=leon2003@turner.name&username=manuel1949"
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a GET method -- MARCHE
http://localhost:4000/api/users/:userID 
Commande : curl http://localhost:4000/api/users/6
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a POST method -- MARCHE
http://localhost:4000/api/users 
Commande : Invoke-RestMethod -Uri http://localhost:4000/api/users -Method Post -ContentType "application/json" -InFile "user.json"

Fichier json associé : 
{
  "user": {
    "email": "milo@gmail.com",
    "username": "milo484"
  }
}
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a PUT method -- MARCHE
http://localhost:4000/api/users/:userID
Commande : Invoke-WebRequest -Uri http://localhost:4000/api/users/6 -Method PUT -ContentType "application/json" -Body (Get-Content -Raw -Path update_user.json)
-----------------------------------------------------------------------------------------------------------------------------------------------------------
– a DELETE method -- MARCHE
http://localhost:4000/api/users/:userID
Commande : Invoke-WebRequest -Uri http://localhost:4000/api/users/6 -Method DELETE
